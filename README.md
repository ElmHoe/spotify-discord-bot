## Features
- add songs to your playlist
- join your vc
- plays song from playlist
- skip

- pause
- resume
- volume increase/decrease
- time into song
- youtube search or add youtube link per vid. for playlist
- invite
- ping
- connections throughout guilds bot is in
- guild count
